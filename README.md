# ครูรักสอน

ระบบเว็บสำหรับครู ใช้จัดการรายวิชา ห้องเรียน บทเรียน ใบงาน แบบทดสอบ การสั่งงาน ตารางสอน ปฏิทิน เวลาเรียน และสรุปคะแนนทั้งเทอม

## ความสามารถหลัก
- Login แยกบทบาท: admin / teacher / student
- ครูหลายคนแยกข้อมูลตามเจ้าของงาน
- นักเรียนเห็นเฉพาะงาน/วิชาที่ถูกสั่งให้ห้องของตัวเอง
- เพิ่ม/แก้ไข/ลบ ผู้ใช้ ห้องเรียน รายวิชา หน่วย บทเรียน ใบงาน แบบทดสอบ ข้อสอบ งานที่สั่ง ตารางสอน และปฏิทิน
- นำเข้านักเรียนจาก Excel: เลขบัตรประชาชน | ชื่อ-สกุล | วันเกิด
- Username นักเรียน = เลขบัตรประชาชน, Password เริ่มต้น = วันเดือนปีเกิด
- ใบงานทั่วไปและใบงานเปตองแบบตารางระยะ/สถานี
- แบบทดสอบออนไลน์ ตรวจคะแนนอัตโนมัติ
- เงื่อนไขจบบทเรียน: เข้าเรียน + ส่งใบงาน + ทำแบบทดสอบผ่าน
- ตารางสอนแสดงคาบปัจจุบัน/คาบถัดไป
- ปฏิทินวันหยุด/กิจกรรม/ไปราชการ
- เช็กเวลาเรียน ขาด ลา มาสาย
- สรุปคะแนนทั้งเทอมและ Export Excel
- UI รองรับมือถือด้วยเมนูแฮมเบอร์เกอร์และตารางแบบการ์ด

## รันระบบ
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python app.py
```

ถ้าพอร์ต 5000 ชน ให้รัน:
```bash
PORT=8000 python app.py
```

## v20 ปรับโครงบทเรียนประจำคาบ
- แยก “งานพิเศษ/รายงาน” ออกจาก “บทเรียนประจำคาบ”
- นักเรียนเห็นบทเรียนจากตารางเรียนและแผนบทเรียนทั้งเทอมเรียงตามวันที่มีเรียนจริง
- ถ้าคาบไม่ได้ผูกบทเรียนไว้ ระบบจะดึงบทเรียนตามลำดับหน่วย/บทเรียนในรายวิชาให้อัตโนมัติ และข้ามวันหยุดตามปฏิทิน
- นักเรียนทำใบงาน/แบบทดสอบจากหน้าบทเรียนได้โดยตรง ไม่ต้องรอครูสั่งงานพิเศษ
- แก้ใบงานเปตองให้เป็นตารางระยะ 6.5/7.5/8.5/9.5 x สถานี 1-5 พร้อมรวมคะแนน
- ปรับการแสดงรูปภาพบทเรียนและภาพใบงานให้เห็นชัดขึ้น
