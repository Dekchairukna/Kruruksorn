# KRURUKSORN Production Fix Pack 1

สิ่งที่ปรับในชุดนี้

1. รวมเมนูระบบบริหารโรงเรียนและ Phase 1–5 ไว้ที่หน้า `/school-erp`
2. ลดเมนูด้านข้างให้เหลือระบบบริหารโรงเรียนเป็นศูนย์กลางเดียว
3. เพิ่มปุ่มย่อ/ขยายเมนูด้านข้างบน Desktop
4. ปรับตารางสอนให้คาบที่เป็นวิชาเดียวกัน เวลาเดียวกัน หลายห้อง แสดงเป็นปุ่มเดียว เช่น `เปิดคาบ ม.1/2,ม.1/3 เช็กชื่อด้วย`
5. เพิ่มหน้าเช็กชื่อรวมหลายห้อง `/attendance/group/<schedule_id>` โดยเรียงรายชื่อนักเรียนตามห้องเลขน้อยก่อน แล้วค่อยเรียงรหัส/ชื่อ
6. ลบไฟล์ขยะจากชุดส่งงาน เช่น `.venv`, `.git`, `__pycache__`, `.DS_Store`, `app.py.bak`

หมายเหตุ: ระบบยังเป็น MVP/Prototype ที่พร้อมทดสอบข้อมูลจริง 1 ห้องหรือ 1 ระดับชั้นก่อน Production เต็มรูปแบบ
