# KRURUKSORN Production Fix Pack 1.1

ปรับเมนูด้านข้างให้ย่อแล้วเหลือเฉพาะไอคอน ไม่ซ่อนเมนูทั้งแถบ

## แก้ไข
- ปุ่มย่อ/ขยาย sidebar ยังอยู่ตำแหน่งเดิม
- เมื่อกดย่อ เมนูซ้ายจะเหลือแถบกว้างประมาณ 76px
- แสดงเฉพาะไอคอนของแต่ละเมนู
- ซ่อนชื่อระบบ ชื่อผู้ใช้ และข้อความเมนู
- เนื้อหาหลักขยับตามขนาด sidebar อัตโนมัติ
- จำสถานะย่อ/ขยายด้วย localStorage
- hover ที่เมนูไอคอนจะมี title ช่วยบอกชื่อเมนู

## ไฟล์ที่แก้
- templates/base.html
- static/css/style.css
