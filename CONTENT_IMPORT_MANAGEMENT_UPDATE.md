# อัปเดตระบบจัดการหน่วย/บทเรียน

ปรับตามคำขอ:

1. นำเข้าหน่วย/บทเรียนแบบอัปเดตได้
   - อัปเดตของเดิม + เพิ่มใหม่
   - คงข้อมูลเดิมไว้ + เพิ่มเฉพาะรายการใหม่
   - ทำทับเฉพาะหน่วยที่ชื่อซ้ำ
   - ล้างหน่วยเดิมทั้งหมดแล้วนำเข้าใหม่

2. จัดการหน่วยง่ายขึ้น
   - เพิ่ม checkbox เลือกหลายหน่วย
   - ลบหน่วยที่เลือกพร้อมกันได้
   - มีปุ่มเลือกทั้งหมด / ยกเลิกเลือก

3. ยืนยันการลบแบบ popup กลางหน้าจอ
   - ไม่ใช้ confirm() ของ browser แล้ว
   - ไม่ต้องเลื่อนขึ้นลงเพื่อกดยืนยัน
   - ใช้ได้ทั้งลบหน่วยเดียว ลบหลายหน่วย และนำเข้า Excel โหมดเสี่ยง
