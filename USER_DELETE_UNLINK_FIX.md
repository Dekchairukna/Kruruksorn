# แก้การลบไอดีครูแบบไม่ผูกข้อมูล

เวอร์ชันนี้ปรับการลบผู้ใช้ role `teacher` ให้ปลดการผูกข้อมูลก่อนลบบัญชีจริง

สิ่งที่ทำตอนลบครู:
- ลบแถวผูกครูกับรายวิชา (`teacher_subject`)
- ลบแถวผูกครูกับห้อง (`teacher_classroom`)
- ตั้ง `subject.teacher_id` และ `classroom.teacher_id` เป็นว่าง
- ตั้งประวัติ `checked_by_id` และ `substitute_for_teacher_id` ใน attendance เป็นว่าง
- ปลด `attendance.schedule_id` แล้วลบตารางสอนของครูคนนั้น
- งานที่ครูเคยสั่งจะโอนไปให้ admin คนแรก เพื่อไม่ให้ข้อมูลการส่งงานหาย

หมายเหตุ: ข้อมูลเช็กชื่อเดิมไม่ถูกลบ เพราะยังผูกกับรายวิชา ห้อง นักเรียน วันที่ และสถานะอยู่
