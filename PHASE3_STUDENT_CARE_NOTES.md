# Phase 3 Student Care Update

เพิ่ม Phase 3 ต่อจาก Phase 2 โดยยังไม่แตะระบบ LINE กลุ่ม

## หน้าใหม่

- `/phase3` ศูนย์รวม SDQ / เยี่ยมบ้าน / ดูแลช่วยเหลือนักเรียน
- `/phase3/sdq` รายการ SDQ และลิงก์กรอกออนไลน์
- `/phase3/sdq/new/<student_id>` ครูกรอก SDQ ให้นักเรียน
- `/sdq/<token>` ลิงก์สาธารณะให้ผู้ปกครอง/นักเรียนกรอก SDQ โดยไม่ต้อง Login
- `/phase3/home-visits` รายการเยี่ยมบ้าน และลิงก์ผู้ปกครองกรอก
- `/phase3/home-visits/new/<student_id>` ครูบันทึกเยี่ยมบ้าน
- `/home-visit/<token>` ลิงก์สาธารณะให้ผู้ปกครองกรอกข้อมูลเยี่ยมบ้าน
- `/phase3/care-records` ระบบเคสดูแลช่วยเหลือนักเรียน
- `/phase3/care-records/new/<student_id>` เปิดเคสดูแลช่วยเหลือ
- `/phase3/care-records/<record_id>/edit` แก้ไข/ปิดเคส

## ตารางใหม่

- `sdq_assessment`
- `home_visit`
- `student_care_record`

ใช้ `db.create_all()` ตอนเริ่มระบบ จึงไม่ลบข้อมูลเก่า

## หลักการ

- SDQ เป็นการคัดกรองเบื้องต้น ไม่ใช่การวินิจฉัย
- ผู้ปกครองและนักเรียนสามารถกรอกผ่านลิงก์เฉพาะรายคน
- ครูยังเป็นคนตรวจสอบและเปิดเคสดูแลเอง
- ไม่ให้ระบบ AI หรือระบบอัตโนมัติแก้ข้อมูลเอง
