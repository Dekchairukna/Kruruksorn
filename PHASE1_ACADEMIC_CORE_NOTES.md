# Phase 1 Academic Core

เพิ่มชุดใช้งานจริงเริ่มต้นสำหรับ KRURUKSORN โดยเน้น 4 งานหลัก

1. ข้อมูลนักเรียน
   - ใช้หน้าห้องเรียนเดิม `/classroom/<id>/students`
   - เพิ่มนักเรียนแบบเร็ว เลือกนักเรียนเดิม ย้ายห้อง แก้ไขข้อมูลพื้นฐานได้

2. เช็กชื่อ
   - ใช้หน้าเช็กชื่อเดิม `/attendance/<subject_id>/<classroom_id>`
   - บันทึก มา/สาย/ขาด/โดดเรียน/ลาป่วย/ไปกิจกรรม
   - เช็กเป็นรายคาบ และมีคะแนนในคาบ

3. คะแนนและเกรด
   - ใช้หน้า `/grades/<subject_id>/<classroom_id>`
   - ตั้งค่าน้ำหนักคะแนน
   - กรอกกลางภาค ปลายภาค จิตพิสัย/คุณลักษณะฯ
   - คำนวณรวมและเกรดอัตโนมัติ

4. ปพ.5
   - เพิ่มหน้าพิมพ์ `/pp5/<subject_id>/<classroom_id>`
   - เพิ่ม export Excel `/pp5/<subject_id>/<classroom_id>/export`
   - เพิ่มปุ่ม ปพ.5 ในหน้าสมุดคะแนนและศูนย์เช็กชื่อ/คะแนน

เมนูใหม่
- เพิ่มเมนู `🎯 Phase 1 วิชาการ`
- หน้า `/phase1` เป็นศูนย์รวมสถานะวิชา/ห้อง พร้อมปุ่มไปทำงานหลักทันที

หมายเหตุ
- ยังไม่ใช่ ปพ.5 แบบละเอียด 100% ตามฟอร์มโรงเรียนทุกช่อง แต่เป็นฉบับใช้งานจริงเบื้องต้นเพื่อพิมพ์/ส่งออก Excel ได้ก่อน
- ไม่เพิ่ม AI ให้แก้ข้อมูลเอง และไม่แตะ KRURUK SPORTS
